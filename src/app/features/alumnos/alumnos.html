Validaciones aplicadas
  • Tipo Documento + Número Documento → Unique Key (no se permite duplicar)
  • Nombres / Apellidos → solo texto, sin caracteres especiales ni números
  • Fecha de Nacimiento → formato de fecha válido
  • numeroDocumento y fechaNacimiento se cifran con AES antes de persistir
